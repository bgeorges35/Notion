T5

Text To Text Transfer Transformer

T5 model can perform several task (with one unique model), such as:

  • Classification
  • Question Answering
  • Machine Translation
  • Summarization
  • Sentiment analysis

Pre-training

We mask a part of the orignal text and create inputs and targets with token indicating where the missing parts are

Architecture

  • Encoder / decoder
  • 12 transformers blocks each
  • 220 million parameters

The architecture is made of 3 parts

  1. Encoder / Decoder:
    • Encoder → encoder transformer block, with full attention
    • Decoder → decoder transformer block, with causal attention
  1. Language model →
    • Take concatenation of input and target as input
    • Use causal attention
  1. Prefix LM → full attention on the left and causal attention on the right
