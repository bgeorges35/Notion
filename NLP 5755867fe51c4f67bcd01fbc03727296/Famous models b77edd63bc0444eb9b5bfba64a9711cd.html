Famous models

We will see a simplified history of some famous NLP model

CBOW

Continuous Bag-of-Words Model

Already seen in 📚Word embedding with NN : Try to predict a missing word in a sentence with a fix sized window. Then use learned weight as words embedding

Problems:

  • Fix window size: we don't use all the context word we could

ELMo

Embeddings from Language Models

ELMo use a bi-directional LSTM (RNN) to predict the missing word.

GPT

Generative Pre-trained Transformer (Open AI)

GPT use a Transformer with only a Decoder

GPT is uni-directional (can't use words after the word to predict as context) because it uses causal attention

⇒ Issue with long term dependency

BERT

Bidirectional Encoder Representations from Transformers

More details in ↔️BERT

Bert is a Transformer with just an encoder

It uses words before and after as context

T5

Text To Text Transfer Transformer

T5 is a transformer with a decoder and en encoder

T5 is capable of performing several task (with only one model 🤯 )

To allow that, T5 uses a "task type" key word placed in front of the real input
