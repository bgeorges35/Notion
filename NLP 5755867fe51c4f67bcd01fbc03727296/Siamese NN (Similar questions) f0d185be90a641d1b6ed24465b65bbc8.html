Siamese NN

Identify similarity between things, with a score.

Application ‚Üí Question duplicate, Handwritten signature, query (similar or tint)

Architecture

Steps to compute cost

  1. Create 2 batches: Put similar sentence on the same line and different one in the same column. A batch is a column
  1. Pass the first batch in the first subnetwork and the second batch in the second subnetwork.

    You get two matrix (v1v_1v1‚ÄãÔªø and v2v_2v2‚ÄãÔªø) of size (batch_size, embedding_size)

  1. Compute the cosine similarity of the two vvvÔªø matrix :

    Each column represent a sentence from v1v_1v1‚ÄãÔªø and each column a sentence from v2v_2v2‚ÄãÔªø. The value in a cell is the similarity between the two corresponding sentences

    The value in the diagonal (similarity between similar exemple) should be higher than the rest

We then have two choice

Improved Triplets Loss

  • Mean negative ‚Üí Mean of values not in the diagonal (negative exemple, orange cell here)
  • Closest negative ‚Üí Value not in the diagonal closest (but smaller) to the diagonal one, in the same row

We then define two new sub-loss by replacing the s(A,N)s(A, N)s(A,N)Ôªø parts or the original loss by the mean negative and the closest ngative

We then add the two sub-loss to get our real loss.
