BERT

Bidirectional Encoder Representations from Transformers

Architecture

  • Multi layer bidirectional transformer
  • Positional embeddings
  • Orange square (ENE_NEN​﻿) → Embedding input words
  • Blue circle → Transformers
  • Green square (TNT_NTN​﻿) → Prediction

BERT_base:

  • 12 layers (12 transformer blocks)
  • 12 attentions heads
  • 110 million parameters

Steps

Pre-training

BERT uses 2 pre-training tasks with unlabelled data (self supervised)

BERT is trained to this two task at the same time

Multi-Mask Language Modeling

or MLM

Next Sentence Prediction

Choose the most likely next sentence

Fine-tuning

  1. Initialise with pre-trained weights
  1. All parameters are fine-tuned (no freeze), using labeled data from the downstream task

To fine tune it on different downstream task, you need to change a bit the input / output

  • MLNI → Multi-Genre Natural Language Inference (tell if 2 sentence are neutral, opposite, ...)
  • NER → Named Entity Recognition
  • SQuAD → Stanford Question Answering Dataset

Input / Output

Input

BERT uses an addition of 3 types of embedding

Output

BERT is trained to its two pre-training task at the same time

So the output must work for both task:

  • CCC﻿ → Used for next sentence prediction
  • TNT_NTN​﻿ → Used to predict masked words. For each word in input, there's an output word. With masked-Language modeling task, only the output of masked words is considered

Objective

It uses Cross Entropy Loss and Binary Loss
