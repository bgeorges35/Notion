Transfer learning (Question Answering)

Transfer learning

Take an already trained model and train it again (but less) on a task a bit different, a "Downsteam task", to adapt it to your specific task

  • visual

    For similar task

    For different task

  • compare with classic training

Advantages

  • Reduce training time
  • Improve prediction
  • Can use smaller datasets

There's 2 method to make transfer learning

Feature-based

We use a already trained model to modify the feature we use (like with Continuous Bag-of-Words)

Fine-Tune

We reuse directly the pre-trained model's weight to make our prediction

With fine-tuning we can add an output layer.

For exemple: We have a pre-trained model to note movies on 3 star and want to use it to note something else on 5 star.

We just have to add an output layer and train only this last layer (and frizzing the other weights)

Data

More data → bigger model → bigger result
